How to manage user access with user permissions

Invite users to access your account with their own logins and limit the level of access they have.
Written by Kirsty McLaughlin, Updated over a week ago

User permissions can only be managed by the organisation administrator - at first, this is the person who created the Appointmentor organisation and as users are added, any user can be an administrator.

There are three levels of user permissions which you can grant to a user of your Appointmentor organisation:

  • An administrator has full access to Appointmentor, all features and can also manage user permissions. This is usually the owner of the organisation.
  • A standard user has access to Appointmentor and all features but cannot manage user permissions. This is usually the manager of the organisation.
  • A restricted user has access to those calendars which are assigned to them by an Administrator. They can make and modify bookings in those calendars and access tasks assigned to those resources. A restricted user can search for a customer to add them to a booking, therefore giving them access to the name, phone number and email address of relevant cusotmers. They do not have access to customers tab, admin, marketing or settings. 

Top tip: if you want to give someone access to a resource's bookings but don't want them to log in to your Appointmentor account, you can connect the resource's Appointmentor calendar to an external calendar e.g. Google or Outlook. Check out this support guide for more details.

How to add a user to your organisation

You must be an Administrator of the organisation to add a user.

  • Click the settings cog
  • Click users 
  • Click add user 
  • Enter the name of the user
  • Enter the email address of the user
  • Select the level of permissions that this user should have
  • If this user will be a restricted user, you should select the resources' calendars which they should have access to
  • Click send invite. 

An invite is sent to the user's email address. The user will then follow the link to accept the invite. If the user already has a user account (this could be on a different Appointmentor account), they will be prompted to log in to that account. If they don't yet have a user account, they will create their password.

This user now has their own login to your Appointmentor organisation.

How to re-send an invitation

  • In the user management page, click on the pencil icon next to the user you have invited
  • Click resend invitation 

How to delete a user

The same process should be followed to delete an existing user or to delete a user who has not yet accepted an invitation:

  • In the user management page, click on the pencil icon next to the user you wish to delete
  • Click delete user or delete invite 
  • Click yes 

How to edit the permissions of a user

  • In the user management page, click on the pencil icon next to the user you wish to edit
  • Select the user permissions which you wish the user to have
  • Click save changes.
